# 001

## Chmod private key

```sh
$ chmod 400 develop.pem
```

## Connect SSH

```sh
$ ssh -i "develop.pem" username@server
```

## Copy files from server to local

```sh
$ scp -i "develop.pem" username@server:file.zip ./file.zip
```

## Zip files on SSH

```sh
$ zip -r workspace.zip wordspace -x '*/.git/*' -x '*/node_modules/*'
```
